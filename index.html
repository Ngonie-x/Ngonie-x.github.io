Work

Photo by Blake Connally on Unsplash

Most of the projects I have worked on are on my github account.

Here are a few:

PassSafe - A django powered password storage application making use of AES encryption to encryption the passwords before saving them to the database.

ShareFiles - An android application made using python that tranfers files between android devices using networking concepts. The backbone of the project is the python sockets library.

QR and Barcode Scanner - A mobile application that scans QR and Bar codes harnessing opencv and pyzbar packages.

AnimeApp - I am a big fan of anime. This application is intended to keep me up to date with my favorite anime by scraping anime websites and downloading information about the anime. It makes use of the BeautifulSoup library and is fully built in python.

Email Download App - I made this application so that I could extract all the information from my email account for processing. It is full built in python and makes use of IMAPClient and pyzmail Libraries.
